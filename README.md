# 🔐 PassForge — Password Generator

A clean, secure, and modern password generator built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- ⚡ Cryptographically secure passwords using `crypto.getRandomValues()`
- 🔠 Toggle: Uppercase, Lowercase, Numbers, Symbols
- 📏 Adjustable password length (6–64 characters)
- 💪 Live strength meter (Weak → Fair → Strong → Excellent)
- 📋 One-click copy to clipboard
- 🔢 Entropy display (in bits)
- 🎨 Dark themed, responsive UI
